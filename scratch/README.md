# scratch/

Working directory for transient files produced while stimmafying a workflow:
`plan.json`, intermediate workflow copies, one-off helper scripts, render-test
outputs, etc.

**Everything in here except this README is gitignored.** Put junk here freely —
it will not be committed. Nothing in this directory is load-bearing; it is safe
to delete the contents at any time.

The `stimmafy` skill writes its `plan.json` and any temporary scripts here by
default.
